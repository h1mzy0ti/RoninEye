Introduction

Welcome to the RoninEye documentation. Learn how to install, configure, and use our API management tools.

Installation

To get started with RoninEye, you need to install our package using npm, yarn, or pnpm. 

npm install @ronineye/core
yarn add @ronineye/core
pnpm add @ronineye/core

Configuration

After installation, you need to configure RoninEye to work with your application. 

// ronineye.config.js
export default {
  apiKey: 'your-api-key',
  endpoint: 'https://api.example.com',
  version: 'v1',
  timeout: 30000,
  retries: 3,
}

Create this configuration file in the root of your project and replace the values with your actual API credentials.

Basic Usage

Here's a simple example of how to use RoninEye in your application: 

import { ApiClient } from '@ronineye/core';
import config from './ronineye.config.js';

const client = new ApiClient(config);

// Fetch data from your API
async function fetchData() {
  try {
    const response = await client.get('/users');
    console.log(response.data);
  } catch (error) {
    console.error('Error fetching data:', error);
  }
}

fetchData();
